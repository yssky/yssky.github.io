System Center Virtual Machine Manager 2016评估版已过期无法打开控制台

发表于 2018-07-02 | 分类于 System Center

System Center Virtual Machine Manager 2016评估版已过期无法打开控制台，如何输入正式版Key激活评估版？

网上方法如下，到底能不能激活为正式版呢？

以管理员身份打开Virtual Machine Manager Command Shell，输入以下命令注册SCVMM为正式版。

Register-SCVMMAccessLicense -ProductKey “XXXXX-XXXXX-XXXXX-XXXXX-XXXXX” -AcceptEULA

使用VMM Powershell输入如上命令报错如下：

此方法无效，因为评估版过期，造成Powershell组件也无法连接VMM管理服务器，也无法使用此条命令激活为正式版。

此条命令是在评估版还没到期时，可以激话为正式版。

正确的解决方法是：卸载SCVMM，保留数据，再重新安装SCVMM。以下是操作方法截图，请按照顺序操作！
