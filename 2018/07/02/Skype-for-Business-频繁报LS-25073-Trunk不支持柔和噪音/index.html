Skype for Business 频繁报LS 25073 Trunk不支持柔和噪音

发表于 2018-07-02 | 分类于 Skype for Business Server

部署Skype for Business Server 2015并启用企业语音，企业内部部署奥科M1000语音网关，一切看起来都工作正常，但SfB前端频繁报事件ID 25073 Trunk不支持柔和噪音。

请按照如下频繁配置AudioCode M1000语音网关相关配置，以支持柔和噪音。

1.AudioCode M1000配置如下项目为Enable

Enable RFC 3389 CN Payload Type :Enable

Comfort Noise Generation Negotiation:Enable

2.配置如下编解码率Silence Suppression :Enable

如上配置AudioCode M1000支持柔和噪音，Skype for Business前端服务器不再报柔和噪音错误。

Skype for Business Server
SCVMM2016群集主机状态需要注意
