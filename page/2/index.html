System Center 2016 Virtual Machine Manager热增加虚拟机内存

发表于 2018-07-02 | 分类于 System Center

ystem Center 2016 Virtual Machine Manager热增加虚拟机内存（modify the memory configuration of a running VM that uses static memory），前提条件：Windows Server 2016 Hyper-v和托管的Windows Server 2016虚拟机，通过System Center 2016 Virtual Machine Manager 虚拟机UI管理界面无法热增加内存（注意：可以通过宿主机Hyper-v管理器热增加内存），需要通过Powershell命令，命令如下：
PS C:\> $VM = Get-SCVirtualMachine -Name “VM01”
PS C:\> Set-SCVirtualMachine -VM $VM -MemoryMB 1024

参考：https://docs.microsoft.com/en-us/system-center/vmm/vm-settings

wsus(Windows Server Update Services)控制台报错:WsusPool应用程序停止

发表于 2018-07-02 | 分类于 WSUS

#故障描述：
wsus(Windows Server Update Services)控制台经常无法打开，报错提示如下：

WSUS 管理控制台无法通过远程 API 连接到 WSUS 服务器。

请确保 Update Services 服务、IIS 和 SQL 正在该服务器上运行。如果该问题仍然存在，请尝试重新启动 IIS、SQL 和 Update Services 服务。

System.Net.WebException — 请求因 HTTP 状态 503 失败: Service Unavailable。

以及如下报错：

无法访问 WSUS 内容目录。
System.Net.WebException: 远程服务器返回错误: (503) 服务器不可用。
在 System.Net.HttpWebRequest.GetResponse()
在 Microsoft.UpdateServices.Internal.HealthMonitoring.HmtWebServices.CheckContentDirWebAccess(EventLoggingType type, HealthEventLogger logger)

解决方法：
1.更新WsusPool应用程序池的“专用内存限制”为合理值，如12G
2.关闭WsusPool应用程序池的“快速故障防护”
3.重新启用WsusPool应用程序池 iisreset
可以使用如下命令快速执行：
“c:\Windows\System32\inetsrv\appcmd.exe” set apppool /apppool.name:wsuspool /recycling.periodicRestart.privateMemory:”12582912″
“c:\Windows\System32\inetsrv\appcmd.exe” set apppool /apppool.name:wsuspool /failure.rapidFailProtection:”false”
“c:\Windows\System32\inetsrv\appcmd.exe” list apppool “wsuspool” /text:*

部署Skype for Business Server Statistics Manager(统计管理器)

发表于 2018-07-02 | 分类于 Skype for Business Server

Skype for Business Server的统计管理器是一个强大的工具，可让您实时查看Skype for Business Server运行状况和性能数据。您可以每几秒钟轮询几百台服务器的性能数据，并在统计管理器网站上立即查看结果。

1.安装Redis-x64-3.0.501，下载地址：https://github.com/MSOpenTech/redis/releases
2.申请自定义证书
3.安装统计管理器侦听器服务 StatsManPerfAgentListener.msi ，下载地址：https://aka.ms/StatsManDownload
4.安装统计管理器网站 StatsManWebSite.msi
5.在Skype for Business前端、边缘、分支等服务器上安装统计管理器代理 StatsManPerfAgent.msi
6.导入Skype for Business服务器拓扑
导出：Get-CsPool | Export-Clixml -Path mypoolinfo.xml
在统计管理器服务器上导入：
cd ‘C:\Program Files\Skype for Business Server StatsMan Listener’

.\Update-StatsManServerInfo.ps1 -CsPoolFile <path to mypoolinfo.xml>

WSUS管理windows 10质量更新（快速安装文件功能）

发表于 2018-07-02 | 分类于 WSUS

由于 Windows 10 的缘故，质量更新比传统的 Windows 更新要大，因为质量更新是累积更新。 为了管理客户端下载类似这些大型更新需要的带宽，WSUS 拥有名为 Express Installation（快速安装文件） 文件的功能。

配置 WSUS 以下载 快速安装文件

打开 WSUS 管理控制台。
在导航窗格中，依次转到你的服务器\选项。
在选项部分中，单击更新文件和语言。
在更新文件和语言对话框中，选择下载快速安装文件

Skype for Business联系人和组备份与恢复命令

发表于 2018-07-02 | 分类于 Skype for Business Server

Skype for Business联系人和组备份与恢复命令：

1.导出备份：Export-CsUserData
例如导出某一个用户联系人和组：
Export-CsUserData -PoolFqdn pool01.yangqs.com -UserFilter zhangsan@yangqs.com -FileName c:\zhangsan.zip

2.导入覆盖恢复：Import-CsUserData
例如导入某一个用户联系人和组并覆盖原有的：
Import-CsUserData -FileName “c:\zhangsan.zip” -PoolFqdn pool01.yangqs.com -UserFilter zhangsan@yangqs.com -v

3.导入更新恢复：Update-CsUserData
例如导入某一个用户联系人和组并更新原有的：
Update-CsUserData -Filename “c:\zhangsan.zip” -UserFilter zhangsan@yangqs.com

参考：https://docs.microsoft.com/en-us/powershell/module/skype/import-csuserdata?view=skype-ps

Skype for Business外呼呼叫记录查询SQL语句

发表于 2018-07-02 | 分类于 Skype for Business Server

Skype for Business外呼呼叫记录查询调用存档数据库LcsCDR视图表VoipDetailsView，此表数据时间基于格林尼治时间，需要在此基础上加8小时东八区(UTC/GMT+08:00)。删除一些不常使用的字段，外呼呼叫记录查询SQL语句如下。

SELECT DATEADD(hour,8, [SessionIdTime]) as [SessionIdTime] ,DATEADD(hour,8, [InviteTime]) as [InviteTime] ,[FromUri] ,[ToUri] ,[FromUriType] ,[ToUriType] ,DATEADD(hour,8, [EndTime]) as [EndTime] ,[FromClientVersion] ,[FromClientType] ,[FromClientCategory] ,[ToClientVersion] ,[ToClientType] ,[ToClientCategory] ,[TargetUri] ,[TargetUriType] ,DATEADD(hour,8, [ResponseTime]) as [ResponseTime] ,[ResponseCode] ,[DiagnosticId] ,[FrontEnd] ,[Pool] ,[FromEdgeServer] ,[ToEdgeServer] ,[IsFromInternal] ,[IsToInternal] ,DATEADD(hour,8, [LastModifiedTime]) as [LastModifiedTime] ,[FromPhone] ,[ToPhone] ,[DisconnectedByUri] ,[DisconnectedByUriType] ,[DisconnectedByPhone] ,[FromMediationServer] ,[ToMediationServer] ,[FromGateway] ,[ToGateway] FROM [LcsCDR].[dbo].[VoipDetailsView] where FromUri = ‘qiangshengyang@yangqs.com’ and ResponseCode = 200 order by SessionIdTime desc

Skype for Business手机APP不能推送消息通知

发表于 2018-07-02 | 分类于 Skype for Business Server

最近在实施本地Skype for Business Server 2015与21v Office 365混合部署，项目实施完成后，用户反馈手机端Skype for Business没有消息推送通知。

在iPhone手机上测试，也没有消息推送通知。在微软工程师协助下，最终诊断结论是：

Skype for Business基于云推送服务MPNS只有国际版，国内世纪互联版不提供。需要再添加一条国际版的联机宿主提供程序。

配置参考链接：https://technet.microsoft.com/zh-cn/library/hh690047(v=ocs.15).aspx

排查步骤及解决方法：

1.使用命令Test-CsMcxPushNotification测试推送服务，失败

2.查看联机宿主提供程序，是国内版

3.查看推送配置，已启用

4.查看推送push.lync.com配置，已配置

5.查看联盟配置，已启用

6.添加国际版推送云服务宿主提供程序

7.查看所有宿主提供程序

8.测试推送服务，成功

总结：国内世纪互联Office365不提供云端的推送通知服务，如果做了与21v的混合部署，需要添加国际版云推送服务

Windows Server 2016部署Skype for Business Server安装组件rewrite_2.0_rtw_x64.msi失败

发表于 2018-07-02 | 分类于 Skype for Business Server

Windows Server 2016部署Skype for Business Server安装组件rewrite_2.0_rtw_x64.msi失败，提示版本IIS不一致，可以通过注册表把IIS版本从10调整为7，即可顺利部署。

参考文章进行修改：https://www.eshlomo.us/rewritemodule-error-with-skype-for-business-on-windows-server-2016/

http://pdhewaju.com.np/2017/02/02/solvedprerequisite-installation-failed-rewritemodule/

当然，也可以使用最新补丁里的rewrite_2.0_rtw_x64.msi 覆盖部署向导调用的准备条件组件。

部署向导调用组件路径：C:\ProgramData\Microsoft\Skype for Business Server\Deployment\cache\6.0.9319.0

新补丁的路径：C:\ProgramData\Microsoft\Skype for Business Server\Deployment\patchcache\6.0.9319.0
