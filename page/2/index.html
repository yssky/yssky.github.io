System Center 2016 Virtual Machine Manager热增加虚拟机内存

发表于 2018-07-02 | 分类于 System Center

ystem Center 2016 Virtual Machine Manager热增加虚拟机内存（modify the memory configuration of a running VM that uses static memory），前提条件：Windows Server 2016 Hyper-v和托管的Windows Server 2016虚拟机，通过System Center 2016 Virtual Machine Manager 虚拟机UI管理界面无法热增加内存（注意：可以通过宿主机Hyper-v管理器热增加内存），需要通过Powershell命令，命令如下：
PS C:\> $VM = Get-SCVirtualMachine -Name “VM01”
PS C:\> Set-SCVirtualMachine -VM $VM -MemoryMB 1024

参考：https://docs.microsoft.com/en-us/system-center/vmm/vm-settings

WSUS管理windows 10质量更新（快速安装文件功能）

发表于 2018-07-02 | 分类于 WSUS

由于 Windows 10 的缘故，质量更新比传统的 Windows 更新要大，因为质量更新是累积更新。 为了管理客户端下载类似这些大型更新需要的带宽，WSUS 拥有名为 Express Installation（快速安装文件） 文件的功能。

配置 WSUS 以下载 快速安装文件

打开 WSUS 管理控制台。
在导航窗格中，依次转到你的服务器\选项。
在选项部分中，单击更新文件和语言。
在更新文件和语言对话框中，选择下载快速安装文件

Skype for Business联系人和组备份与恢复命令

发表于 2018-07-02 | 分类于 Skype for Business Server

Skype for Business联系人和组备份与恢复命令：

1.导出备份：Export-CsUserData
例如导出某一个用户联系人和组：
Export-CsUserData -PoolFqdn pool01.yangqs.com -UserFilter zhangsan@yangqs.com -FileName c:\zhangsan.zip

2.导入覆盖恢复：Import-CsUserData
例如导入某一个用户联系人和组并覆盖原有的：
Import-CsUserData -FileName “c:\zhangsan.zip” -PoolFqdn pool01.yangqs.com -UserFilter zhangsan@yangqs.com -v

3.导入更新恢复：Update-CsUserData
例如导入某一个用户联系人和组并更新原有的：
Update-CsUserData -Filename “c:\zhangsan.zip” -UserFilter zhangsan@yangqs.com

参考：https://docs.microsoft.com/en-us/powershell/module/skype/import-csuserdata?view=skype-ps

Skype for Business外呼呼叫记录查询SQL语句

发表于 2018-07-02 | 分类于 Skype for Business Server

Skype for Business外呼呼叫记录查询调用存档数据库LcsCDR视图表VoipDetailsView，此表数据时间基于格林尼治时间，需要在此基础上加8小时东八区(UTC/GMT+08:00)。删除一些不常使用的字段，外呼呼叫记录查询SQL语句如下。

SELECT DATEADD(hour,8, [SessionIdTime]) as [SessionIdTime] ,DATEADD(hour,8, [InviteTime]) as [InviteTime] ,[FromUri] ,[ToUri] ,[FromUriType] ,[ToUriType] ,DATEADD(hour,8, [EndTime]) as [EndTime] ,[FromClientVersion] ,[FromClientType] ,[FromClientCategory] ,[ToClientVersion] ,[ToClientType] ,[ToClientCategory] ,[TargetUri] ,[TargetUriType] ,DATEADD(hour,8, [ResponseTime]) as [ResponseTime] ,[ResponseCode] ,[DiagnosticId] ,[FrontEnd] ,[Pool] ,[FromEdgeServer] ,[ToEdgeServer] ,[IsFromInternal] ,[IsToInternal] ,DATEADD(hour,8, [LastModifiedTime]) as [LastModifiedTime] ,[FromPhone] ,[ToPhone] ,[DisconnectedByUri] ,[DisconnectedByUriType] ,[DisconnectedByPhone] ,[FromMediationServer] ,[ToMediationServer] ,[FromGateway] ,[ToGateway] FROM [LcsCDR].[dbo].[VoipDetailsView] where FromUri = ‘qiangshengyang@yangqs.com’ and ResponseCode = 200 order by SessionIdTime desc
